node 是什么
node 是一个基于 Chrome V8 引擎的 JavaScript 运行环境
网上已经有很多很多的解释和说明了,这里不就行跟多的赘述,就尝试用简短的话语去进行描述
1.node的核心是一个运行环境,他实现了ECMAScript规范,然后他还可以有自己的一套api,他和浏览器不一样,浏览器有dome,但是node没有 2.node可以调用底层的api,比如文件系统,网络请求,进程管理.每一个不同的语言,都有自己的实现规范和运行环境,实现规格意味着你的代码可以怎么写,运行环境意味着你有什么能力.而实现规范只是意味着你的代码可以怎么写,看他的设计思路是怎么回事 3.基本上所有的高级语言,他都是有自己的规范和运行环境的 4.node这里先介绍介绍node的原生功能
4.1 os模块,操作系统模块,可以获取操作系统的信息
4.2 fs模块,文件系统模块,可以读取文件,写入文件,删除文件
4.3 path模块,路径模块,可以获取路径的信息
4.4 http模块,网络请求模块,可以发送网络请求
4.5 url模块,url模块,可以解析url
4.6 querystring模块,查询字符串模块,可以解析查询字符串
4.7 zlib模块,压缩模块,可以压缩和解压文件
4.8 crypto模块,加密模块,可以加密和解密文件
