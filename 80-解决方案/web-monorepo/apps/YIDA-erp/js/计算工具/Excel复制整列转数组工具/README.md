# Excel 复制整列转数组工具

这是一个用于将 Excel 中复制的整列数据转换为 JavaScript 数组格式的工具集。

## 📁 文件说明

### 核心文件

- **`a.js`** - 主要转换脚本，将文本文件转换为数组格式
- **`a.txt`** - 输入文件（从 Excel 复制的数据）
- **`b.txt`** - 输出文件（转换后的 JavaScript 数组格式）
- **`code.js`** - 数组比较工具，用于找出两个数组之间的差异
- **`c.txt`** - 结果文件（可选）

## 🚀 使用方法

### 1. Excel 数据转数组

#### 步骤 1：准备数据

1. 在 Excel 中选择需要转换的整列数据
2. 复制选中的数据（Ctrl+C）
3. 将数据粘贴到 `a.txt` 文件中（每行一个值）

#### 步骤 2：运行转换脚本

```bash
node a.js
```

#### 步骤 3：查看结果

转换后的数组会自动保存到 `b.txt` 文件中，格式如下：

```javascript
[
	'值1',
	'值2',
	'值3',
	...
]
```

### 2. 数组比较功能

`code.js` 文件包含两个数组 `a` 和 `b`，可以用于：

- 找出数组 a 中存在但数组 b 中不存在的值
- 数组去重和比较

#### 使用方法

```bash
node code.js
```

## ✨ 功能特点

### a.js 脚本功能

- ✅ 自动读取 `a.txt` 文件
- ✅ 按行处理数据
- ✅ 自动去除空白行
- ✅ **自动去重**（使用 Set 数据结构）
- ✅ 转换为 JavaScript 数组格式
- ✅ 所有值都作为字符串处理（带引号）
- ✅ 输出格式化，便于阅读和复制

### code.js 脚本功能

- ✅ 支持两个数组的比较
- ✅ 找出差异值
- ✅ 字符串格式处理
- ✅ 性能优化（使用 Set 数据结构）

## 📝 示例

### 输入文件 (a.txt)

```
600000074511588
129105617195649
129105719012733
600000074511588
129105617752325
```

### 输出文件 (b.txt)

```javascript
['600000074511588', '129105617195649', '129105719012733', '129105617752325'];
```

注意：重复的值 `600000074511588` 已被自动去重。

## ⚙️ 配置说明

### a.js 配置

- **输入文件**：`a.txt`（当前目录）
- **输出文件**：`b.txt`（当前目录）
- **去重**：自动启用
- **格式**：JavaScript 数组格式，带缩进

### code.js 配置

- **数组 a**：积加数据
- **数组 b**：ERP 数据
- **比较方式**：字符串比较
- **输出**：控制台输出差异结果

## 💡 使用场景

1. **Excel 数据导入**：将 Excel 中的列数据快速转换为 JavaScript 数组
2. **数据去重**：自动去除重复值
3. **数据比较**：比较两个数据源的差异
4. **代码生成**：快速生成数组代码用于开发

## 📌 注意事项

1. 确保 `a.txt` 文件存在于脚本同目录下
2. 输入数据每行一个值
3. 所有值都会被转换为字符串格式（带引号）
4. 重复值会自动去重
5. 空行会被自动跳过
