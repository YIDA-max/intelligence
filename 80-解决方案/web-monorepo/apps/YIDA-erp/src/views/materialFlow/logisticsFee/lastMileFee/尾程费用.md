# 尾程费用（需求摘要）

> 完整 PRD 以产品/飞书为准；此处为开发对照摘要。

## 模块与菜单位置

- **物流 - 物流费用 - 尾程费用**

## 数据来源

- 自发货订单，**订单状态 = 已发货**
- 定时清洗：每小时增量；每日 02:00 同步前一天有更新的订单

## 列表

- **维度**：平台单号 + 系统订单号
- **排序**：发货时间 **降序**

### 主要字段（列表）

| 字段                         | 说明                                                           |
| ---------------------------- | -------------------------------------------------------------- |
| 平台单号                     | 自发货订单                                                     |
| 平台渠道                     | 自发货订单                                                     |
| 店铺站点                     | 如「店铺名称:US」                                              |
| 系统订单号                   | 自发货订单                                                     |
| 发货时间                     | 自发货订单                                                     |
| 仓库单号 / 运单号 / 跟踪号   | 自发货订单                                                     |
| 物流商                       | 三方仓 → 服务商；平台物流 → 物流商；转 FBA→ 固定「Amazon FBA」 |
| 发货仓 / 物流渠道 / 目的国家 | 自发货订单                                                     |
| 预估费用(原币)               | 物流比价试算总费用 + 币种                                      |
| 实际费用(原币)               | 用户录入费用项之和 + 币种                                      |
| 费用差额                     | 实际 − 预估；币种不一致时按实际费用币种                        |
| 汇率                         | 发货时间对应月份                                               |
| 实际费用录入时间             | 用户录入时间                                                   |
| 创建时间                     | 首次清洗进列表                                                 |
| 更新时间                     | 记录最近修改时间                                               |

## 录入实际费用

- **不可改**：请款池按「平台单号 + 系统订单号 + 费用项」查询为 **已请款**，或录入时间超过 **3 个月**

### 录入行字段

物流商（三方仓服务商）、费用承担主体（法人主体）、费用项、入账方式、费用分摊方式（实重 KG / 体积 CBM）、费用金额（≥0，四位小数）、币种、结算方式（日结/月结/双月结）、备注（0–100）

同一费用项仅一行；费用项币种需一致否则报错。

### 预览分摊

按 SKU：数量、尺寸、重量、总体积、总重量；分摊单价、分摊总价；前 N−1 行保留 2 位小数，最后一行倒挤；合计等于订单总费用。

## 导入 / 导出

- **导入**：存在则覆盖（未请款且未超期）；已请款或超期报错（可部分成功）
- **导出**：按勾选与筛选；无勾选无筛选则导出全部（列表字段 / 实际费用两类）

## 筛选与搜索

- 发货仓、物流渠道、物流商、平台店铺（平台与站点联动）、目的国家、费用项、费用差额（支持运算符筛选）、录入状态（已录入/未录入）、发货时间、实际费用录入时间
- 搜索：平台单号、系统订单号、仓库单号、运单号、跟踪号（单条模糊、批量精确等按 PRD）

## 操作日志

类型示例：新增、录入实际费用、导入实际费用、确认分摊等；内容附 JSON 详情。
