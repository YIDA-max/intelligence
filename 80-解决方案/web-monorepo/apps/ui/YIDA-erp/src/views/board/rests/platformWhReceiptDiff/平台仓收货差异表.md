## 4.4 平台仓收货差异表

原型链接：

放置位置：报表-其他-平台仓收货差异表

列表展示维度：按平台仓货件单维度展示数据

用父子级列表展示，即一个货件单展开会有多个 SKU 信息

### 1.列表字段说明

| 列表                                         | **字段**                                                | **说明**       |
| -------------------------------------------- | ------------------------------------------------------- | -------------- |
| 父级                                         | Shipment ID                                             | 展示货件单单号 |
| 平台                                         | 展示平台名称                                            |                |
| 仓库代码                                     | 取货架单仓库代码                                        |                |
| 货件单状态                                   | 展示货件单状态                                          |                |
| 调拨单号                                     | 根据 shipment ID=平台入库单找到仟易系统的调拨单号       |                |
| 如果找不到就为空                             |                                                         |                |
| 关联状态                                     | （1）如果找不到关联的调拨单，则关联状态=未关联          |                |
| （2）如果找到关联的调拨单，则关联状态=已关联 |                                                         |                |
| 子级                                         | MSKU                                                    | 货架单 SKU     |
| SKU                                          | 取调拨单该 shipment ID=平台入库单&MSKU 对应的 SKU 编码  |                |
| 找不到就为空                                 |                                                         |                |
| 产品名称                                     | SKU 对应的产品名称                                      |                |
| 找不到就为空                                 |                                                         |                |
| 收货数量                                     | 取货件单收货数量                                        |                |
| 发货数量                                     | 取调拨单该 shipment ID=平台入库单对应的 MSKU 的发货数量 |                |
| 找不到就为空                                 |                                                         |                |
| 差异数量                                     | =收货数量-发货数量                                      |                |
| 差异数量=0，黑色字体                         |                                                         |                |
| 差异数量＜ 0，红色字体                       |                                                         |                |
| 差异数量＞ 0，绿色字体                       |                                                         |                |

### 2.筛选说明

| **筛选项** | **说明**                                   |
| ---------- | ------------------------------------------ |
| 平台       | 下拉搜索多选，数据来源于仟易平台表         |
| 关联状态   | 下拉单选，已关联，未关联，默认选择全部状态 |
| 时间筛选   | 可以按货件单创建日期                       |

### 3.搜索说明

| **搜索维度** | **说明**                                |
| ------------ | --------------------------------------- |
| Shipment ID  | Shipment ID                             |
| 调拨单号     | 支持按调拨单号搜索，单个模糊，批量精确  |
| SKU          | 支持按 SKU 编码搜索，单个模糊，批量精确 |

### 4.操作说明

| 操作       | **说明**                                   |
| ---------- | ------------------------------------------ |
| 更新货件单 | 点击更新货件单数据                         |
| 导出       | 点击导出，支持勾选导出和按筛选导出所有字段 |
